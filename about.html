隐私与安全承诺

我们深知信任的价值。以下是我们对您隐私的庄严承诺：

  • 数据零记录 (Zero Logging): 我们承诺，绝不记录、存储或分析您通过本服务代理的任何请求内容或响应数据。您的对话内容、代码片段或任何业务数据完全属于您自己。
  • API 密钥安全: 您的 API 密钥在传输和存储过程中均经过行业标准的高强度加密处理，确保只有您自己能够访问。
  • 纯粹的技术中转: 我们的服务本质是一个纯粹的技术数据包转发器。您的数据仅在内存中进行必要的处理和转发，绝不会在我们的服务器硬盘上停留。
  • 匿名化统计: 为了优化服务质量（例如判断哪些节点需要扩容），我们可能会收集完全匿名的、聚合性的使用元数据，这些数据无法与任何特定用户关联。
